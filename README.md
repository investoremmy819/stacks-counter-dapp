# Stacks Counter DApp

A simple decentralized app built on the Stacks blockchain that demonstrates:
- Writing a Clarity smart contract
- Connecting a frontend using @stacks/connect
- Interacting with a contract using @stacks/transactions

## Features
- Increment a counter stored on-chain
- Read the current counter value
- Wallet connection using Stacks Wallet

## Tech Stack
- Clarity (smart contracts)
- JavaScript
- @stacks/connect
- @stacks/transactions

## Project Structure

contracts/
counter.clar
frontend/
index.html
app.js

## Getting Started

1. Install Stacks Wallet (Hiro)
2. Deploy the contract to Stacks Testnet
3. Open `frontend/index.html` in your browser
4. Connect wallet and interact with the contract
